# hud_server_time
HUD Server Time — MoonLoader-скрипт для SA:MP с удобным HUD-таймером и меню /taimenu. Показывает серверное время, поддерживает обратный отсчет, паузу и повтор таймеров, настройку текста, шрифта, цвета, обводки и позиции прямо в игре. Красиво, гибко и удобно для повседневной игры и ивентов.
